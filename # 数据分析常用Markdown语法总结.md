## 代码块与公式
### 插入代码块(Python)
```Python
import pandas as pd
# 读取数据
data = pd.read_csv("sales_data.csv")
# 计算销售额均值
avg_sales = data["销售额"].mean()
print(f"平均销售额：{avg_sales:.2f}万元")
```
* 代码功能：读取销售数据并计算平均销售额
### 插入数学公式
1.均值公式：$\bar{x} = \frac{1}{n} \sum_{i=1}^{n} x_i$；  
2.方差公式：$\sigma^2 = \frac{1}{n} \sum_{i=1}^{n} (x_i - \bar{x})^2$；
* 用于计算数据的均值和方差，是数据分析中描述性统计的核心公式

# 学生成绩数据分析
---
## 一、数据概况
本次分析选取了某班级30名学生的春季学期期末成绩数据，涵盖语文、数学、英语三门主科。数据无缺失值，整体呈近似正态分布，为后续分析提供了可靠基础。
 本次分析的主要目标包括：
* 计算各学科的平均分、最高分与最低分
* 统计各分数段的学生人数分布
* 找出班级整体成绩的优势与薄弱环节
---
## 二、数据分析结果
### 各科成绩统计

| 学科 | 平均分 | 最高分 | 最低分 | 及格人数 |
| :---: |:---: |:---: |:---: |:---: |
|语文|80|98|50|24|
|数学|88|100|54|27|
|英语|77|89|44|20|
从表格可以看出**数学**的*平均分以及及格率*为三科最高，**英语***两极分化*最为明显，最低分仅44，是班级最薄弱的科目。
---
### 成绩分布计算
用平均分公式衡量班级整体的平均水平
$\bar{x}=\frac{1}{n}\sum_{i=1}^{n} x_i$
* 公式说明：该公式为**算术平均数计算公式**，其中$\bar{x}$代表平均分，$n$为学生总数，$x_i$为第$i$名学生的成绩，用于反映数据的集中趋势。
同时，我们使用Python代码快速计算并验证了各学科的平均分：
 ```python
 import pandas as pd
 # 读取成绩数据
 data = pd.read_csv("scores.csv")
 # 计算各学科平均分
 avg_scores = data[["语文", "数学", "英语"]].mean()
 print("各学科平均分：\n", avg_scores)
 ```
 * 代码功能：读取成绩数据文件，计算并输出三门学科平均分，用于验证手动计算结果的准确性。
